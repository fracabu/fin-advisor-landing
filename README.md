<a name="english"></a>
## :gb: English

### Overview

An elegant financial advisor landing page built with Vite, TypeScript, and Tailwind CSS. Professional design for wealth management and financial consulting services.

### Features

- **Professional Design** - Trust-inspiring aesthetics
- **Service Showcase** - Highlight financial services
- **Testimonials** - Client success stories
- **Contact Forms** - Easy consultation booking
- **Responsive Layout** - All devices supported

### Quick Start

```bash
git clone https://github.com/fracabu/fin-advisor-landing.git
cd fin-advisor-landing
npm install
npm run dev
```

Access at `http://localhost:5173`
